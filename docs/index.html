Description

Ontologia para o sistema Studynest, uma plataforma mobile de colaboração acadêmica.

                 Esta ontologia modela usuários, disciplinas, grupos de estudo, materiais e mensagens.

                 This work is distributed under Creative Commons Attribution License CC BY 4.0.
